# RaspberryPi-DHT11
Programas ejemplo de como conectar el sensor de temperatura y humedad DHT11 a una Raspberry Pi.

## Software Necesario
- Libreria de Adafruit Python DHT, disponible en https://github.com/adafruit/Adafruit_Python_DHT
